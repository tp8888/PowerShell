# 🏢 Active Directory PowerShell Toolkit

This directory contains labs, scripts, and documentation focused on automating Active Directory administration and security auditing. 



### 🧪 Laboratory Curriculum

| Lab # | Title | Objective | Status |
| :--- | :--- | :--- | :--- |
| **01** | [Environment Verification](./01-Environment-Verification.md) | Validate RSAT and AD module connectivity. | ✅ |
| **02** | [Bulk User Creation](./02-Bulk-User-Creation.md) | Automate onboarding via CSV import. | ✅ |
| **03** | [Privileged Group Audit](./03-Privileged-Group-Audit.md) | Identify and report members of high-risk groups. | 🟡 |
| **04** | [Stale Account Cleanup](./04-Stale-Account-Cleanup.md) | Identify and disable inactive user accounts. | ⚪ |
| **05** | [OU Restructuring](./05-OU-Restructuring.md) | Automated deployment of Organizational Units. | ⚪ |
| **06** | [Password Policy Audit](./06-Password-Policy-Audit.md) | Verify compliance with enterprise password standards. | ⚪ |
| **07** | [User Deprovisioning](./07-User-Deprovisioning.md) | Create a "Kill Switch" for rapid account lockout. | ⚪ |
| **08** | [GPO Inventory](./08-GPO-Report.md) | Export list of GPOs and their linked locations. | ⚪ |
| **09** | [AD Health Check](./09-AD-Health-Check.md) | Test DNS and Domain Controller replication. | ⚪ |
| **10** | [Security Log Parsing](./10-Security-Log-Parsing.md) | Use PowerShell to detect Brute Force events. | ⚪ |

### 🛠️ Core Skills Mastered
* Active Directory PowerShell Module
